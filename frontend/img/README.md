# Preview image for link sharing

Place your **Open Graph / Twitter Card** preview image here:

- **Filename:** `og-preview.png` (or `.jpg`)
- **Recommended size:** 1200Ã—630 px (aspect ratio 1.91:1)
- **Usage:** When someone shares your site link (e.g. on Slack, Twitter, LinkedIn), this image is shown in the preview.

If you use a different filename, update the `og:image` and `twitter:image` meta tags in `index.html` and `blog.html` to match.
